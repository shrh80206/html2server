## 建構方法

先到 https://deno.com/deploy 註冊

然後到 https://dash.deno.com/projects 建立專案

接著授權 deno 使用你的某個 github 之 deno 專案

deno 會給你一個隨便的專案網址名稱，像是 https://whole-frog-10.deno.dev/

接著你可以改成自己想要的 xxx.deno.dev 名稱，但 xxx 不能是別人已經用過的

## 換你了
