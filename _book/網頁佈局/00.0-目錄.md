### **第一部分：HTML 和 CSS 基礎**
1. **HTML 概述與結構**
   - HTML 標籤與結構
   - 頁面骨架：`<html>`, `<head>`, `<body>`

2. **CSS 基礎**
   - CSS 語法與選擇器
   - 顏色、字型、背景與邊框
   - 基本排版：字體大小、行高、字距等

3. **CSS 屬性與簡單排版**
   - `display`, `position`, `float`, `clear`

   - 簡單的文本對齊與布局

5. **Flexbox 佈局**

6. **Grid 佈局**
   - CSS Grid 基礎概念
