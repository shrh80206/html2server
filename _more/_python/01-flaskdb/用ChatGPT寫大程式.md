# 用 ChatGPT 寫大程式

我掌握到用 ChatGPT 幫我寫大程式的技巧了 ...

第一種是

先叫 ChatGPT 寫主程式，然後再把那些還沒寫的函數一個一個貼給他

第二種方法是

先寫出所有函數原型，然後再把那些函數一個一個貼給他

然後 ...

就寫完了

拿去跑跑看，有錯再處理 ...

## 叫 ChatGPT 寫個 B+Tree 是可以的

只是能不能正確跑還沒測

只要叫他先寫 C 的 header 

然後再一個函數一個函數貼給他就行了

...

結果編譯雖然一開始有錯誤

但是補上一些 函數原型之後，竟然直接可以跑 ...
